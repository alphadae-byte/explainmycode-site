# ExplainMyCode.tech

Site vitrine pour l'API ExplainMyCode.

## Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Vercel (hosting)

## Installation

```bash
npm install
npm run dev
```

## Déploiement

Push sur GitHub, connecter à Vercel, déployer automatiquement.

## Structure

```
app/
├── page.tsx                          # Home
├── resultat/page.tsx                 # Analysis results
├── mentions-legales/page.tsx
├── cgu/page.tsx
├── politique-de-confidentialite/page.tsx
└── contact/page.tsx
```
